# React blog
It is coded with React, Nodejs, Typeorm. Server is ts, client is js.

Example .env file.

```
DB_HOST=localhost
DB_USER=root
DB_NAME=blog
DB_PORT=3306
DB_PASS=""

SERVER_PORT=3001

AWS_SECRET=""
AWS_ACCESS=""
AWS_REGION="us-east-2"
AWS_BUCKET="my-bucket"
```
